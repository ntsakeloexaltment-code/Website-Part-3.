# 💄 Beauty Glow Studio | Official Website

**Beauty Glow Studio** is a professional beauty and wellness studio based in Johannesburg, South Africa. This is the full, responsive official website — featuring all pages, consistent soft pink branding (no dark pink), booking system, contact map, and enquiry forms.
